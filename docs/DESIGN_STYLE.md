# Visual style — one paragraph

edu.ninitux is a documentation-density brutalist site in the same family as ninitux.com: dark slate canvas, single arctic-blue accent, system sans + monospace, flat fills with thin 8 %-opacity rules, no gradients, no shadows on layout containers, no decorative illustration. Section headings are short uppercase eyebrows prefixed with `#` in the accent color — the page reads like a well-typeset `README.md`. Cards are bordered rectangles, not raised surfaces; hover means a slightly stronger border, not a shadow. The mascot (white penguin, dark on light theme via `filter: invert`) appears once in the header brand mark and that's it — never as a decoration, never animated. Copy is terse, technical, lowercase where it can be (`# devops`, `+ kubernetes`); marketing voice is forbidden. Everything serves reading speed and trust: a senior engineer should land on a topic, scan the analogy in two seconds, pick the concept they care about, copy a working command, answer one quiz, and leave.
