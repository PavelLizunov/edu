07

CommonЧастые questionsвопросы.

// click to expand
Why "per-process" and not the usual "all traffic"?Почему "по процессам", а не "весь трафик"?

Because most of your traffic is fine on the open net. Your bank, your Slack, your printer, the cloud storage on which your work lives — these should not take the long way around just because you wanted Telegram or a torrent client to use a tunnel. Routing by app means you get the tunnel exactly where you wanted it and pay no latency anywhere else.

Потому что большая часть трафика прекрасно живёт в открытой сети. Банк, Slack, принтер, облако с рабочими файлами — им не нужно крутиться через тоннель только из-за того что вы хотели Telegram или торрент пустить через прокси. Маршрутизация по приложениям даёт тоннель именно там где вы хотели — без задержек везде ещё.

Does it really have no telemetry?Реально нет телеметрии?

Yes. Don't take my word for it — the code is GPL-3.0, you can grep it in 30 seconds (see the Telemetry section). The only outgoing requests are the four listed there, none of which carry usage data.

Да. Не верьте на слово — код под GPL-3.0, grep-нуть можно за 30 секунд (см. раздел Телеметрия). Исходящие запросы только те четыре, что перечислены выше, и ни один из них не несёт usage-данных.

Where do the ≈25k free configs come from?Откуда берутся ≈25k бесплатных конфигов?

From a public scrape of openly-shared VLESS endpoints, refreshed every six hours. They are not curated — quality varies, anything can be down at any time, and you should treat them as found change. The Free tab is for when you have nowhere else to point a tunnel; bring your own server when you can.

Это публичный скрейп открыто-расшаренных VLESS endpoint, обновляется раз в 6 часов. Не курируется — качество разное, любой сервер может лежать, относиться к ним как к найденной мелочи. Free — это когда некуда направить тоннель; свой сервер всегда лучше.

Does it work on a corporate network?Будет ли работать в корпоративной сети?

Sometimes. VLESS+Reality fingerprints as a regular browser TLS handshake which is usually fine; DPI-heavy networks may still block based on flow analysis. Zapret (Windows-only, bundled) helps for the most common cases. If your corp has its own MITM root certificate, no tunnel will get you out — and you should not try.

Иногда. VLESS+Reality маскируется под обычный TLS, что обычно прокатывает; сети с тяжёлым DPI могут блокировать по flow-анализу. Zapret (Windows, в комплекте) помогает в типичных случаях. Если у корпорации свой MITM-сертификат — никакой тоннель не выпустит, и не стоит пытаться.

Does it need root / admin all the time?Нужен ли постоянный root / админ?

No. Linux uses POSIX capabilities so only the TUN-bind needs them — no full-root daemon. macOS writes a one-time sudoers entry at install. Windows needs admin because the TUN driver does; the app itself does not run as SYSTEM.

Нет. Linux использует POSIX capabilities — только TUN-bind их требует, full-root демона нет. macOS пишет одну sudoers строку при установке. Windows требует админа из-за TUN-драйвера; само приложение не работает как SYSTEM.

Mobile support?Поддержка мобилки?

Not yet. iOS and Android are on the roadmap (see roadmap) but not in 2026 H1. For now, mobile users can use a sing-box-compatible client and import the same subscription URL.

Пока нет. iOS и Android в планах (см. роадмап), но не в первой половине 2026. Сейчас на телефоне — sing-box совместимый клиент с той же subscription URL.

Can I trust the auto-updater?Можно ли доверять авто-апдейтеру?

It pulls release manifest from api.github.com, downloads the binary, verifies SHA-256 against the value in the signed manifest, swaps in place. If you don't want it, disable it in Settings → Updates; you'll still get notified.

Тянет манифест с api.github.com, скачивает бинарник, проверяет SHA-256 против значения из подписанного манифеста, подменяет на месте. Не хотите — выключите в Настройках → Обновления; уведомления всё равно будут.

Does this support [my favourite protocol]?Поддерживает ли [мой любимый протокол]?

If sing-box supports it, this does: VLESS, TUIC, Hysteria2, Shadowsocks, Trojan, plain SOCKS5, plain HTTP. WireGuard works as an output too. Drop a JSON, click Apply, done.

Если sing-box поддерживает — поддерживает и это: VLESS, TUIC, Hysteria2, Shadowsocks, Trojan, обычный SOCKS5, HTTP. WireGuard как outbound тоже работает. Кидаете JSON, жмёте Apply, готово.
